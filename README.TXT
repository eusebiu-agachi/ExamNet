Proiect : ExamNet

Subiect : 
- Definirea unui format XML pentru reprezentarea unor teste formate din
intrebari grila si enunturi de probleme, respectiv pentru preluarea solutiilor
- Crearea unor servicii pentru transmiterea testelor, preluare rezultatelor
- Optional: crearea unei interfete Web
- Similar cu examinarea implementata pe platforma pbinfo

Membrii echipei : Agachi Eusebiu (partea de interfata cu utilizatorul) + Benchia Razvan (partea de server)

Tehnologii utilizate :
- Pentru crearea interfetei cu utilizatorului :
  - Ca mediu de lucru : Android Studio
  - Ca limbaj de programare : Kotlin
- Pentru definirea design-ului aplicatiei, s-au utilizat fisiere .xml
- Pentru partea de server :
  - Ca mediu de lucru : IntelliJ IDEA
  - Ca limbaj de programare : Java
  - Ca framework-uri utilizate : Spring Boot
  
 Pe langa cerintele obligatorii, s-au implementat si functionalitati suplimentare, precum inregistrarea/ autentificarea in aplicatie, dar si o fereastra de utilizator, unde 
 clientul isi poate verifica datele din aplicatie (nume de utilizator, adresa de email, scorul obtinut in aplicatie).
